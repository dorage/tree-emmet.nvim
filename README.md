# Tree-Emmet

re-implemenation of emmet for neovim with Tree-sitter

## Features

### Basic Actions
- [ ] Expand Abbreviation
- [-] Balance
    - [v] jsx
    - [ ] html
- [-] Go to Matching Pair
    - [v] jsx
    - [ ] html
- [ ] Wrap with Abbreviation
- [ ] Go to Edit Point
- [ ] Select Item
- [ ] Toggle Comment
- [-] Split/Join Tag
    - [v] jsx
    - [ ] html
- [ ] Remove Tag
- [-] Merge Lines
    - [v] jsx
    - [ ] html
- [ ] Update Image Size
- [ ] Evaluate Math Expression
- [ ] Increment/Decrement Number
- [ ] Reflect CSS Value
- [ ] Encode/Decode Image to data:URL
